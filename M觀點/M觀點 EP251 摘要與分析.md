# M觀點 EP251 摘要與分析

## 重點摘要

此簡報文件綜合分析了「M觀點」第251集的核心內容，涵蓋三大主題：財經、社會觀察與創作者經濟。首先，文件深入剖析軟銀（SoftBank）近期清倉輝達（Nvidia）股票的決策，並提供一個評估輝達未來估值的詳細分析框架。其次，探討了知名投資人彼得·提爾（Peter Thiel）關於年輕世代為何傾向社會主義而非資本主義的觀點，核心歸因於學貸與高房價使他們無法參與資本遊戲。最後，從創作者的角度分析台灣頂級YouTuber「Joeman」宣布其標誌性系列節目《Joe是要對決》停更的事件，探討了內容創作者面臨的倦怠、「內耗」以及對「常態性內容聖杯」的迷思。

## 軟銀（SoftBank）清倉輝達（Nvidia）股票之分析

### 事件背景與官方解釋

在2025年11月11日發布的財報中，軟銀證實已於10月份將其持有的輝達股票全數出清，總計賣出3,210萬股，價值58.3億美元，平均賣出價約為每股180美元。軟銀官方強調，此舉並非看空輝達，而是出於財務規劃需要，籌集現金以支持其今年的重大投資案，特別是與OpenAI的合作，包括入股以及支持「星際之門」（Stargate）超級電腦計畫。

為籌措資金，軟銀不僅出售了輝達持股，還賣出了價值92億美元的美國電信商T-Mobile股份，顯示其正積極調整資產配置以投入其認為未來潛力更大的領域。

### 歷史回顧：2019年的錯誤決策

市場對軟銀此次操作的反應相對平淡，部分原因在於輝達股價在過去三年已實現約13倍的驚人漲幅。然而，軟銀曾在2019年犯下過一次重大的投資失誤。當時，為籌資投資WeWork與Uber等共享經濟公司，軟銀在輝達股價因加密貨幣挖礦需求崩盤而處於低點時，賣出了價值40億美元的持股。

這筆當時看似獲利了結（約賺30多億美元）的交易，事後被證明是巨大的錯誤。孫正義本人也承認：「如果我們當時繼續持有，我們將能多賺超過1500億美元。」（"if we had held on, we would have made over 150 billion"）。當年賣出的40億美元股票，如今價值已超過1500億美元。

### 2025年情境分析：與2019年的差異

本次賣股與2019年的情境存在顯著差異，或可避免重蹈覆轍：

1. **賣出時機**：2019年是在輝達股價低點時賣出，而此次是在股價經歷大幅上漲後的相對高點。
2. **投資標的**：2019年的資金投向了商業模式備受質疑的WeWork與Uber，而此次的核心投資標的OpenAI，被視為AI領域的明日之星。

此決策可解讀為，軟銀認為將資金從當紅的輝達轉移至潛力巨大的OpenAI，即使OpenAI目前估值已高達5000億美元，其未來的回報率可能更高。

### 輝達估值的核心框架

講者指出，評估輝達的估值極其複雜，困難之處不在於計算公式（如DCF模型），而在於對未來市場情境的準確判斷。所有估值模型中的參數（如成長率、市佔率、毛利率）皆取決於對市場動態的預測。因此，要評估輝達的價值，必須回答以下兩個核心問題及其子問題：

#### 關鍵問題一：未來十年AI算力需求的成長速度

1. **子問題1：AI如何提升人類生產力？**
   - 必須深入理解AI在日常生活與工作中如何實質性地改變與提升效率。
   - 需觀察使用者（個人與企業）將任務交給AI處理的頻率與廣度是否持續增加。講者以自身為例，認為其當前使用AI的量體相比兩年前已增加了20至30倍。
2. **子問題2：生產力提升如何轉換為商業模式？**
   - 生產力的提升必須成功轉化為商業價值，即有消費者或企業願意為此付費。
   - 若終端用戶不願付費（無論是直接訂閱或透過廣告等間接模式），算力需求的增長將無法持續。

#### 關鍵問題二：輝達在市場競爭中的定位

1. **子問題1：算力需求的成長與單位成本的下降**
   - 儘管總算力需求上升，但單一算力任務的成本卻在持續下降。這源於技術進步，例如從FP16/FP8精度推論轉向更高效的FP4，以及模型演算法的優化（如小型模型能達到過去大型模型的效果）。
   - 因此，3000%的算力需求成長，不等於3000%的硬體營收成長。
2. **子問題2：ASIC晶片帶來的競爭壓力**
   - 來自Google（TPU）、Amazon（Inferentia）、Microsoft（Maia）等雲端服務巨頭的自研ASIC晶片將是輝達未來的主要競爭者。
   - 評估的關鍵在於，輝達的通用性GPU平台（CUDA生態系）與其所主張的「總體擁有成本」（Total Cost of Ownership, TCO）優勢，能否在未來十年持續抵禦專用ASIC的挑戰。
3. **子問題3：綜合評估下的輝達成長率與毛利率預測**
   - 綜合以上因素，才能預測輝達在未來十年的AI GPU營收成長率與毛利率。例如，若市佔率從目前的90%下降，其定價能力和毛利率也將隨之下滑。

### 一個簡化的估值推演範例

講者提供了一個**高度簡化且假設性的推演**，旨在展示上述框架的應用思路，**不應視為投資建議**：

| 估值參數               | 假設與推演過程                                               |
| ---------------------- | ------------------------------------------------------------ |
| **總算力需求成長**     | 假設十年後，全球70%的人口將其80%的資訊需求交由AI處理，且個人總需求量翻倍，推算出個人應用算力需求成長約19倍（1800%）。加上企業應用及多模態需求，**粗估十年總算力需求成長為3000%**。 |
| **GPU營收成長**        | 考慮到硬體效率提升（如FP4）與模型演算法進步，將3000%的算力需求成長折算為硬體營收成長。**假設折算係數為3，則GPU/ASIC市場十年營收成長約為1000%**。 |
| **市佔率變化**         | 假設十年後，輝達的市佔率從目前的90%**下降至60%**，而ASIC佔比上升至40%。 |
| **輝達營收年化成長率** | 根據上述假設，計算得出輝達未來十年的營收年化成長率（CAGR）約為**21%**。考慮通膨因素後，可上調至**25%**。 |
| **毛利率與營業利益率** | 隨著競爭加劇和市佔率下滑，輝達的毛利率可能從目前的75-80%**回落至60-65%**的較合理水平。營業利益率預計可維持在24-25%。 |
| **估值結論**           | 在此假設情境下，輝達的合理股價區間大約在**200至250美元**之間。 |

講者最後強調，「模糊的正確比準確的錯誤更好」，估值的核心在於建立一個能應對未來不確定性的思考框架，而非追求單一精確的數字。

## Peter Thiel 對年輕世代反資本主義的觀察

### 背景與核心論點

隨著紐約市選出帶有社會主義色彩的市長，Peter Thiel在2020年寫給Meta（當時為Facebook）高層的一封信再度引起關注。信中，Thiel探討了為何美國年輕人日益傾向社會主義，其核心論點是：**年輕世代因學貸與高房價兩大因素，被排除在資本主義的遊戲之外。**

- **學貸壓力（Student Debt）**：年輕人一出社會就背負沉重學貸，處於「負資本」狀態，薪水主要用於還債而非積累資本，無法參與投資（如購買股票、基金）。
- **高昂房價（Housing Prices）**：即使還清學貸，高昂的房價也使他們難以購買房產。買房是普通人參與資本遊戲、利用槓桿實現資產增值的主要途徑之一。

當年輕人在資本主義的遊戲規則中沒有任何利益（stake）時，他們自然傾向於反對這個「別人的遊戲」。

### 深入探討：學貸與高等教育

Thiel認為美國高等教育體系存在嚴重問題，許多人支付了高昂的學費，卻未能學到足以償還債務的技能。

- **學費高昂**：即使是州內居民就讀本州公立大學，一年學費加生活費也約需4萬美元；私立名校（如哈佛、史丹佛）則高達每年9至10萬美元。
- **投資失衡**：念AI等高需求科系的畢業生能輕易償還學貸，但念哲學、性別研究、非洲研究等社會科學學位的畢業生，往往面臨就業困難與低薪，導致學貸成為終身負擔。
- **Thiel的結論**：「太多人去念大學，什麼都沒有學到，卻背了一屁股沉重的債。」

講者補充，學貸問題的根本解決方案在於銀行應基於科系的未來還款能力進行風險評估，而非無差別放貸。

### 「理所應得」（Entitlement）的心態批判

講者認為，部分年輕人抱持一種「理所應得」的心態，認為自己「值得」或「配得」接受高等教育，而忽略了其作為一項昂貴投資的本質。這種「社會欠我的」心態，是導致做出不理智財務決策（如背負無法償還的學貸）的深層原因。

### 深入探討：房價問題

房價問題比學貸更為棘手，其根源在於經濟學基本原理：

- **價值不均**：不同地段因其創造的商業價值不同，價格自然有高低。
- **都市化趨勢**：資源與機會向都市集中，導致人口持續湧入，推高都市房價，這是追求效率的必然結果。

講者認為，政府能做的有限，核心應圍繞「增加供給」與「降低（特定區域的）需求」：

1. **增加供給**：放寬建築法規限制，鼓勵建商蓋房；透過稅收政策（如對空屋課徵高持有稅，但對出租住宅減免）鼓勵屋主將閒置房屋投入租賃市場。
2. **降低需求**：在核心都市圈以外創造新的就業中心，引導人口分流。例如，台灣的南科與高雄設廠計畫，有助於分散竹科的居住壓力。

## 創作者觀察：《Joe是要對決》停更事件

### 事件概要與創作者的「內耗」

台灣頂級YouTuber「Joeman」宣布其經營五年、流量穩定的標誌性系列節目《Joe是要對決》將暫停更新。Joeman表示主要原因是**創作倦怠**。

講者從創作者角度分析，這種倦怠源於嚴重的「內耗」：

- **重複性的疲勞**：長期製作同類型內容（特別是美食主題），即使是山珍海味也會感到厭膩。
- **表演的痛苦**：當創作者對內容失去熱情時，仍需在鏡頭前表現出興奮與享受，這種「表演」會大量消耗內心能量，導致痛苦加劇。
- **缺乏成就感**：創作者除了追求流量與收入，也需要來自於「創造新事物」的自我肯定。持續重複相同的模式，會剝奪這種成就感，最終導致創作力枯竭。

### 聖杯的迷思：常態性內容的挑戰

《Joe是要對決》的模式，是許多創作者夢寐以求的「常態性內容聖杯」，其特點是：

1. **製作相對容易**：不需絞盡腦汁想題材，有固定模式可循。
2. **流量穩定**：能持續吸引觀眾。
3. **可長期執行**：題材看似取之不盡。

然而，講者指出，**真正的「常態性內容聖杯」並不存在**。任何成功的內容模式都有效性，通常一至兩年便會遇到瓶頸。原因包括：

- **觀眾會看膩**。
- **創作者會做膩**，最終導致自我耗竭。
- 在YouTube等演算法平台，平台會不斷推送新創作者，舊有內容模式面臨的競爭壓力更大。

### 創作者的選擇與反思

講者以自身為例，說明其從YouTube轉向Podcast，是為了擺脫演算法的壓力，從而可以更「任性」地製作自己真正感興趣的內容。這種選擇帶來的好處是不易倦怠，但代價是成長較慢，也較難獲得爆炸性的成功。

結論是，創作者的人生無法「又要且要」。追求極致成功往往伴隨著巨大的壓力與內耗；而追求內心平衡與創作自由，則可能需要接受較為平緩的發展曲線。每個創作者都必須在其中做出自己的選擇。
