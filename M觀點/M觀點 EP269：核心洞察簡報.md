# M觀點 EP269：核心洞察簡報

## 執行摘要

本簡報綜合分析了「M觀點」第269集直播的核心議題，涵蓋了科技產業的重大結盟、商業模式的轉變以及投資哲學的辯論。主要洞見如下：

- **Apple與Google的AI戰略結盟**：為了解決在人工智慧領域的明顯落後，蘋果（Apple）選擇與谷歌（Google）達成一項多年期合作協議，將採用Google的Gemini模型技術來打造其「蘋果基礎模型」（Apple Foundation Model）。此舉是蘋果在無法憑藉一己之力追趕領先者的情況下，所做出的務實且最佳的策略選擇，旨在確保iPhone在未來AI競局中不致落後。此合作不僅鞏固了Google在AI模型領域的領導地位，也有效阻擋了OpenAI等競爭對手進入蘋果生態系的機會。
- **特斯拉FSD全面轉向訂閱制**：特斯拉（Tesla）宣布其全自動輔助駕駛（FSD）功能將取消一次性買斷方案，全面改為月費訂閱制。此轉變被視為一個完全正確的商業決策，使FSD從傳統的硬體附加功能銷售模式，進化為現代化的軟體即服務（SaaS）模式。此舉不僅能為特斯拉帶來更穩定、長期的現金流，也降低了消費者的入門門檻，增加了使用彈性，對公司與用戶而言是雙贏局面。
- **主動與被動投資之爭的省思**：近期在社群媒體上關於主動投資與被動投資的激烈辯論，被認為是一場無謂且重複的爭論。報告指出，兩種投資方式各有利弊，並無絕對優劣。對於絕大多數散戶而言，被動指數化投資是更為穩妥且勝率較高的選擇。然而，對於有能力且願意投入研究的投資者，也不應抹煞其透過主動投資追求超額報酬的可能性。最佳策略是深入了解兩種方法的本質，並根據個人情況找到最適合自己的配置，例如採用混合策略或從少量資金開始驗證自己的主動投資能力。

## detailed analysis

### 一、Apple與Google的AI戰略結盟

此合作標誌著蘋果在AI策略上的重大轉向，從堅持自研轉為務實地尋求外部合作，以應對日益激烈的技術競賽。

#### 背景：蘋果的AI困境與現實考量

長久以來，蘋果在AI領域的進展始終緩慢，其語音助理Siri的表現多年來未有實質性突破，常受外界詬病。隨著大型語言模型的快速發展，蘋果在2023年發布的「Apple Intelligence」計畫亦未能及時推出具競爭力的產品，使其落後的態勢更加明顯。

報告分析，蘋果之所以難以追趕，主要受限於以下因素：

- **企業文化與結構**：作為一家由專業經理人（Tim Cook）領導的公司，蘋果的文化較為保守，難以像創辦人主導的公司（如Meta）一樣，為挖角頂尖AI人才而開出顛覆內部薪資結構的天價。
- **成本控制思維**：蘋果以其精密的供應鏈成本控制著稱，這使其難以輕易投入數百億美元巨資採購GPU以進行模型訓練。
- **市場壓力**：競爭對手如三星（Samsung）和Google的Pixel手機已將AI功能作為核心賣點。若iPhone在AI能力上持續缺席，其核心業務將面臨嚴重威脅。

基於上述現實，蘋果承認在短期（3至5年內）無法單獨追上AI領先者，因此與Google的合作成為其現階段最合理的策略選擇。

#### 推測的合作模式與選擇Google的原因

雖然官方未公布細節，但報告推測此次合作的模式極可能如下： Google將其Gemini模型作為一項智慧財產權（IP）授權給蘋果。蘋果基於此核心技術，利用自身的用戶數據、軟硬體整合優勢進行後續訓練（Post-training）與微調（Fine-tuning），最終打造出一個深度整合並能高效操控iOS裝置的「Apple Foundation Model」。此模式類似於微軟（Microsoft）與OpenAI的合作關係。

蘋果選擇Google而非OpenAI等其他廠商，是基於多重戰略考量：

| 考量因素           | 分析內容                                                     |
| ------------------ | ------------------------------------------------------------ |
| **既有夥伴關係**   | 儘管在手機市場是競爭對手，蘋果與Google在搜尋引擎等領域有長期且穩固的合作默契，彼此信任基礎較高。相較之下，OpenAI的野心可能對蘋果的硬體生態構成潛在威脅。 |
| **模型技術適配性** | 蘋果的AI架構涵蓋端點、私有雲和公有雲三層。Google的Gemini模型系列完整，特別是為行動裝置設計的**Gemini Nano**，完美符合蘋果對端點小型模型的需求，這是OpenAI目前所缺乏的。 |
| **雲端基礎設施**   | 合作聲明中提及「Google的雲端技術」，暗示蘋果可能在其私有雲中採用Google的**TPU**（Tensor Processing Unit）。由於Gemini模型原生於TPU開發，此舉能達到最佳的效能與成本效益，符合蘋果的經營理念。 |
| **合作價格**       | 傳聞中的年授權費為10億美元，對蘋果而言是小數目，顯示Google在此次合作中更看重戰略意義而非短期財務收益。 |

#### 戰略影響評估

- **對蘋果**：此為一次「不得不為」的明智之舉，有效化解了未來的重大經營風險。但這也可能降低蘋果未來獨立研發頂尖大型模型的決心，其研發重心或將轉向如何將外部模型與自身生態系做最佳整合。
- **對Google**：這是一次重大勝利。不僅能將其AI模型部署到全球最龐大的高端智慧手機用戶群中，獲得寶貴的實戰數據與反饋，還能帶來雲端業務收入，並成功阻止了競爭對手進入蘋果生態。
- **對市場輸家**：OpenAI與馬斯克的xAI在此次競逐中落敗，錯失了一個觸及全球數億用戶的關鍵分發渠道。

### 二、特斯拉FSD全面轉向訂閱制

特斯拉將FSD的商業模式由一次性買斷改為純訂閱制，這項決策被視為符合產業趨勢且極具遠見的舉措。

#### 商業模式的必然演進

過去，FSD的銷售模式類似於汽車的硬體選配（如天窗、音響），與車輛本身綁定。然而，現今所有特斯拉車輛出廠時均已配備FSD所需硬體，其啟用與否僅是軟體層面的開關。因此，將其視為軟體服務並採用訂閱制是合乎邏輯的演進。

此轉變帶來多重商業優勢：

1. **穩定的經常性收入**：訂閱制能提供可預測且持續的現金流，優於一次性銷售的收入波動。
2. **降低用戶門檻**：每月99美元的費用遠低於8000美元的買斷價格，能吸引更多用戶嘗試使用，擴大市場滲透率。
3. **更高的長期價值**：以車輛平均10至15年的使用壽命計算，訂閱制的總收入將遠超過一次性買斷費用。報告估算，在美國市場僅需訂閱7年，總費用就已超越買斷價。

#### 消費者價值與彈性

對消費者而言，訂閱制同樣帶來好處：

- **消費彈性**：用戶可根據需求隨時訂閱或取消，例如在長期出國時暫停付費。
- **賬號可攜性**：FSD服務與車主賬號綁定，而非車輛。當車主更換新特斯拉時，其駕駛習慣與數據可無縫轉移，持續享受個人化的駕駛體驗。
- **高性價比**：報告指出，每月約100美元的費用，即可獲得一位「隨傳隨到」的專屬司機，能處理日常通勤、自動停車等任務，大幅節省時間，價值遠超其價格。

#### 對台灣市場的呼籲

報告特別指出，台灣在自動駕駛技術的開放進程上相對落後。講者呼籲台灣政府應以更前瞻的思維擁抱新科技，逐步開放相關法規。他認為，對計程車等產業的衝擊是不可避免的趨勢，與其消極防堵，不如採取漸進式開放，給予產業長達數年的緩衝與轉型時間，這才是更負責任的做法。

### 三、主動投資與被動投資之爭的省思

針對近期社群平台上的激烈辯論，報告提出應回歸理性，並為投資者提供了一套務實的思考框架。

#### 爭議的核心：一場無謂的辯論

主動投資（試圖挑選個股以超越市場表現）與被動投資（追蹤市場指數以獲取平均報酬）之間的爭論被比喻為「北部粽與南部粽之爭」，兩者各有特色與愛好者，硬要分出高下並無意義。

#### 對不同投資者的建議

報告強調，不存在一體適用的最佳方法，關鍵在於「認識自己」與「了解工具」。

- **對絕大多數散戶**：歷史數據與學術研究明確顯示，超過九成的散戶在進行主動投資時，長期績效落後於市場大盤。因此，**被動指數化投資**是更為穩健、安全的基礎策略。
- **對有志挑戰者**：儘管困難，市場中確實存在少數能夠長期戰勝大盤的投資者。對於有能力、有熱情、願意投入時間研究的人，不應完全否定主動投資的價值。

#### 推薦的漸進式實證路徑

報告建議採用一種務實的驗證方法，而非盲目投入：

1. **初期配置**：將絕大部分資金（如90%）投入被動指數型基金，確保獲取市場基本回報。
2. **小額試驗**：用小部分資金（如10%）進行主動投資。
3. **績效檢驗**：定期（如每年）檢視主動投資部分的績效是否超越大盤。
4. **逐步加碼**：如果該部分連續數年（如2-3年）穩定跑贏大盤，並且成功**經歷至少一次大型空頭市場的考驗**而未被淘汰，才可考慮逐步增加主動投資的資金比例。

此方法讓投資者在風險可控的前提下，客觀評估自己是否具備超越市場的能力。講者以自身經驗為例，他也是在經歷了早期市場的慘痛教訓後，才逐步找到適合自己的長線投資策略，並將投資組合配置為50%主動、50%被動的混合模式。

最終結論是，投資者應將時間精力專注於研究與建立適合自己的體系，而非浪費在無益的線上爭論中。
